# Документаци я по API

## Транзакции

### Создать

POST запрос на эндпоинт ``api/transactions`` в формате JSON:

```json
[
    {
        "date": "05.01.2026 18:12:00",
        "name": "Кроссовки",
        "category": [
            1,
            2,
            3
        ],
        "amount": "15055.5"
    },
    {
        "date": "01.01.2026 23:00:00",
        "name": "Пальто",
        "category": [
            1
        ],
        "amount": "200055.75"
    }
]
```

Обязательно передавать массив. Если нужно добавить 1 элемент, то массив с одним элементом.

Если Категорий с указанными id не существует - транзакции будут добавлены, категории у них будут прописаны только существующие.

Цена в виде строки, разделитель - точка.

### Просмотреть все транзакции

GET запрос на эндпоинт ``api/transactions``

Также возможно фильтровать по категорим и/или по дате

```
api/transactions?category_id=2&date=01-05-2026
```

## Категории

### Создать

POST запрос на эндпоинт ``api/transaction-categories`` в формате JSON:

```json
{
    "categories": [
        "Одежда",
        "Обувь",
        "Еда",
        "Развлечения",
        "Разное"
    ]
}
```

### Просмотреть все категории

GET запрос на эндпоинт ``api/transaction-categories``
