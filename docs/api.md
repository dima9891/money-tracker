# Документаци я по API

## Транзакции

### Создать

POST запрос на эндпоинт ``api/transactions/create`` в формате JSON:

```json
[
    {
        "date": "05.01.2026 18:12:00",
        "name": "Кроссовки",
        "category": [
            1,
            2,
            3
        ],
        "amount": "15055.5"
    },
    {
        "date": "01.01.2026 23:00:00",
        "name": "Пальто",
        "category": [
            1
        ],
        "amount": "200055.75"
    }
]
```

Обязательно передавать массив. Если нужно добавить 1 элемент, то массив с одним элементом.
Если Категорий с указанными id не существует - транзакции будут добавлены, категории у них будут прописаны только существующие.
Цена в виде строки, разделитель - точка.

### Просмотреть

POST запрос на эндпоинт ``api/transactions/list``
